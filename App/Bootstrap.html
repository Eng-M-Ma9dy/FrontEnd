< About Bootstrap > 

* Bootstrap :

    - Utilities: Group of Class's in CSS File Ready For Use During i write the HTML Code for Style
    - Components: Group of Ready Templates For Use, By Copy HTML Code From BS Site and Paste in HTML

    - BS Working Via Link (CSS and JS) Files of Bootstrap by Your HTML Sheet

    1. Online Link CSS and JS File
        - Open Bootstrap Web Site
        - Open Docs Of Bootstrap > Getting started > Introduction, and Follow Steps.

    2. Offline Link CSS and JS File
        - Open Bootstrap Web Site
        - Open Docs Of Bootstrap > Getting started > Download
        - Under Section Compiled CSS and JS, Press on Download Button
        - Extract This File : Via WinRAR Program
        - Take (bootstrap.min.css) File From CSS Folder
        - Take (bootstrap.bundle.min.js) File From JS Folder
        - Set Two File In The Same Project Folder, and Link Two Files in Your HTML Sheet

* Font Awesome :

    - I Use Font Awesome For Add Icons in My Web Site

    - Font Awesome Working Via Link (CSS) Files of Font Awesome by Your HTML Sheet

    - Download Link CSS File
        - Open FontAwesome Web Site
        - Open Start With Free > Other Ways To Use Section > Download > Free For Web
        - Extract This File : Via WinRAR Program
        - Take (all.min.css) File From Folder
        - Set This File In Folder by Name webFont and Set This Folder in The Same Project Folder, and Link This Files in Your HTML Sheet

        - Open Icons Tap Search About Any Icon Name as Facebook, 
        - Take Code Of This Icon Copy and Pest in Your Web Site
