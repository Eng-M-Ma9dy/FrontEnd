     > npm i typescript

    2. Create New File index.ts

    3. For Convert File From TS to JS After Code Finished
    - Open Terminal (CMD)
    > tsc index.ts

    4. VS-Code Will Be created index.js File From TS File

    5. link New index.js file in HTML Sheet
